# Student Info App

A React application for managing and displaying student information.

## Features

- **Components**: HeaderComponent and StudentComponent with props
- **Routing**: Home and Students pages with navigation
- **API Integration**: Fetches data from JSONPlaceholder API
- **State Management**: Uses React hooks for state management
- **Error Handling**: Displays loading and error states

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually http://localhost:5173)
